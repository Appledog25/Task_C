About Us

HouseMaterial

This eCommerce website is created by the student interns team in 2021 consisting of four members during their internship program. The website will be user-friendly. Easy to access for sellers and buyers. Clean display interface which will help users to easily handle the website. A fully responsive web application can be browsed from any device. This eCommerce web application will have every facility for buyers and sellers. It will contain a user-friendly interface, be well organized, will save a lot of time for seller & buyer, instant purchase, payment gateways, more importantly, it will be a well-managed web application.
